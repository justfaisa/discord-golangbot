# Discord GoLang Bot
Multiple function Discord Bot built in **Go**.

# Features
- Advanced Music Command
 
# Install
You need to have installed in your system 
    **go>1.7** and **ffmpeg>3.0**

```sh
# Install Discord Bot
go get -u github.com/justfaisa/discord-golangbot
```

# Run the Bot
```sh
# Run the Bot
go run main.go
```
